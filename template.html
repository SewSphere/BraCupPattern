Bra Cup Patterns

This project aims to digitize the bra cup drafting technique from the "How To Make A Bra 2" blog post by Mark Garbarczyk (now accessible only via the Wayback Machine).

These cup sizes are not standard. "Size 4" corresponds to a UK size 34B.

Each pattern is an SVG file measured in millimeters. These files can be easily imported into any vector graphics application, such as Adobe Illustrator or Affinity Designer.
